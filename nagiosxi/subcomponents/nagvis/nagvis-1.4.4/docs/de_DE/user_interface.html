NagVis 1.4 User-Interface

Diese Seite soll die unterschiedlichen Ansichten des NagVis-User-Interfaces beschreiben. Die Pfade sind relativ zum NagVis-root-Verzeichnis.

URLBeschreibung
NagVis Frontend
index.phpDies ist eine Weiterleitung zu nagvis/index.php (NagVis-Frontend)
nagvis/index.phpDies ist der Map-Index, die Hauptseite von NagVis. Auf der Hauptseite sind alle Maps aufgeführt. Sie können eine Map ausschließen, indem Sie die show_in_lists-Option auf 0 setzen.
nagvis/index.php?map=<name>Dies ist die Map-Ansicht. In diesem Modus wird die angegebene Map angezeigt und im konfigurierten Intervall aktualisiert (refreshtime im global-Abschnitt der NagVis-Hauptkonfiguration).
nagvis/index.php?infoDies ist eine Seite mit Support-Informationen, die Sie mitliefern können, wenn Sie Probleme mit NagVis haben.
Rotationen
nagvis/index.php?rotation=<name>Dies ist der Map-Rotationsmodus. In diesem Modus wird die erste angegebene Map angezeigt (Rotation muss in der Hauptkonfigurationsdatei definiert werden). Nach einem definierten Intervall wird die nächste angegebene Map geladen usf.
nagvis/index.php?rotation=<name>&rotate=0Dies ist der Rotationsmodus. Die Rotation wird durch den angegebenen Wert gestoppt. In diesem Modus werden die Maps automatisch aktualisiert. Es kann sinnvoll sein, eine Rotation zu stoppen, wenn Sie etwas Interessantes auf der Map gefunden haben. Die Rotation kann aktiviert werden, wenn Sie den Wert auf 1 setzen oder den Rotationswert löschen.
Automap
nagvis/index.php?automap=1Dies ist der URL zum neuen Automap-Feature (seit NagVis 1.3). Details finden Sie auf der Automap-Seite
WUI (NagVis-Konfiguration)
config.phpDies ist eine Weiterleitung zu wui/index.php (Konfigurations-Interface)
wui/index.phpDies ist die Willkommen-Seite des NagVis-Konfigurations-Interfaces (WUI). Rechtsklicken Sie auf dem Hintergrund-Bild, um das WUI-Menü zu öffnen.
